# agent-design-docs

AIエージェント設計思想ドキュメント

Zenn Book「そのAIエージェント、モデル更新のたびに壊れていませんか？――設計の"補助輪"を外す5つの判断軸」の付録として公開しています。

---

## ファイル構成

```
agent-design-docs/
├── agent-design-philosophy.md   ← 人間向け：設計思想ドキュメント
└── agent-design-review/
      └── SKILL.md               ← AI向け：設計レビュースキル
```

---

## 使い方

### 人間向け：agent-design-philosophy.md

プロジェクト開始前に一度通読して、設計の判断軸を自分の中に入れるためのドキュメントです。

- AIエージェント設計の5つの判断軸
- マルチエージェント固有の落とし穴
- 仕様書への落とし込みテンプレート
- 設計レビューチェックリスト

### AI向け：agent-design-review/SKILL.md

仕様書・要件定義書をClaudeに渡すと、5軸でレビューして問題点と改善案を返すスキルファイルです。

Claudeのスキル機能に登録して使います。
